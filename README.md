# 🏠 RoomScape

RoomScape is an AI-powered web application that transforms 2D floor plans into interactive 3D visualizations. Users can upload their room layouts, generate realistic 3D renders using AI, and manage their designs with persistent cloud storage.

---

## 🚀 Features

* 🖼️ Upload 2D floor plans (images)
* 🤖 AI-powered 3D visualization generation
* 💾 Save projects with user-specific cloud storage (Puter KV)
* 🌐 Image hosting for persistent access
* 🗂️ Project history dashboard
* 🔁 Revisit and edit previously generated designs
* 🔐 User-based data isolation

---

## 🛠️ Tech Stack

* **Frontend:** React + Vite + TypeScript
* **Backend (Serverless):** Puter Workers
* **Storage:** Puter KV + File System
* **AI Integration:** Puter AI (txt2img)
* **Routing:** React Router

---

## ⚙️ How It Works

1. User uploads a 2D floor plan
2. Image is processed and sent to AI
3. AI generates a 3D visualization
4. Images are hosted and stored
5. Project data is saved in KV storage
6. User can revisit projects anytime
